# 🎓 Agendador de Bancas

Sistema inteligente de agendamento de defesas acadêmicas.

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173

## ☁️ Deploy no Vercel (recomendado)

1. Suba este repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e faça login com sua conta GitHub
3. Clique em **"Add New Project"**
4. Selecione o repositório `agendador-de-bancas`
5. Clique em **"Deploy"** — pronto! ✅

O Vercel detecta automaticamente que é um projeto Vite + React e configura tudo sozinho.

## 🛠️ Tecnologias

- React 18
- Vite 5
- CSS puro (sem dependências de UI)
